# QuickPost Privacy Policy

This repository hosts the Privacy Policy page for QuickPost application.

## GitHub Pages Setup

This site is configured to be deployed automatically to GitHub Pages using GitHub Actions.

### Enable GitHub Pages

To enable GitHub Pages for this repository, follow these steps:

1. Go to the repository settings on GitHub
2. Navigate to **Settings** â†’ **Pages**
3. Under **Build and deployment**, select:
   - **Source**: GitHub Actions
4. The site will be automatically deployed when changes are pushed to the `main` or `master` branch
5. Once deployed, the site will be available at: `https://[username].github.io/[repository-name]/`

### Manual Deployment

You can also manually trigger the deployment:
1. Go to the **Actions** tab in the repository
2. Select the "Deploy to GitHub Pages" workflow
3. Click "Run workflow"

## Files

- `index.html` - Privacy Policy page
- `.github/workflows/deploy-pages.yml` - GitHub Actions workflow for automatic deployment
